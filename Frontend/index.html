"Can you hear/see me okay?" - This is a great way to check if the other person can hear and see you clearly.

"It's great to see you!" - This is a friendly and welcoming phrase that can help set a positive tone for the call.

"How's your day/week been?" - This is a good way to start a conversation and catch up with the other person.

"Thanks for taking the time to talk today." - This is a polite way to show your appreciation for the other person's time.

"I'm sorry, could you repeat that?" - If you didn't quite hear what the other person said, it's okay to ask them to repeat themselves.
